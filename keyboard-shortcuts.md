|-------------------------|  :-----------------: |
|  Quick Navigation       | `⌘` + `P`            |
|  Command Palette:       | `SHIFT` + `⌘` + `P`  |
|  Sublime Console:       | `^` + `\``           |
|   Switch Project        | `^` + `⌘` + `P`      |
