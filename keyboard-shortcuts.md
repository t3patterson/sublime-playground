##Some keyboard shortcuts

|  Shortcut Type          |     Keys             |
|-------------------------|  :-----------------: |
|  Quick Navigation       | `⌘` + `P`            |
|  Command Palette:       | `SHIFT` + `⌘` + `P`  |
|  Sublime Console:       | `^` + ```           |
|   Switch Project        | `^` + `⌘` + `P`      |
|   Search for text chunk | `⌘` + `R`            |
|   Code Folding          |`⌘` +  `⌥` + `[`      |
|   Toggle tabs           |`⌘` +  `⌥` + `[`      |
